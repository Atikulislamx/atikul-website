## Overview

This repository contains the complete source code for the official website of **Atikul Islam Rabbi**, a Social Media Security Specialist and Founder & CEO of Cyber Infinity.

The website serves as a central hub for professional information, educational resources, case studies, blog articles, and client communication.

This project is intentionally built using **HTML, CSS, and JavaScript** without frameworks, making it lightweight, fast, easy to understand, and simple to maintain.
