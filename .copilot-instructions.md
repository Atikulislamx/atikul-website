# GitHub Copilot Instructions

## Project Overview

This repository contains the official personal website of **Atikul Islam Rabbi**, a Social Media Security Specialist and Founder & CEO of Cyber Infinity.

The website should present a professional, trustworthy, and modern online presence while remaining lightweight, fast, and easy to maintain.

---

## Technology Requirements

Always use:

* HTML5
* CSS3
* Vanilla JavaScript (ES6)

Do not introduce:

* React
* Vue
* Angular
* Next.js
* TypeScript
* jQuery
* Bootstrap
* Tailwind CSS
* Build tools or package managers unless explicitly requested.

The project must work as a static website hosted on GitHub Pages.

---

## Development Principles

When generating code:

* Prefer simple and maintainable solutions.
* Reuse existing CSS and JavaScript whenever possible.
* Avoid duplicate code.
* Keep HTML semantic and accessible.
* Write clean, readable, and well-organized code.
* Optimize for performance and SEO.

---

## HTML Standards

Every page should include:

* Semantic HTML
* Responsive viewport
* Meta description
* Open Graph metadata
* Twitter Card metadata
* Canonical URL
* Proper heading hierarchy
* Accessible navigation
* Footer

---

## CSS Standards

Use:

* Mobile-first responsive design
* Reusable utility classes
* Consistent spacing
* Professional typography
* Smooth but subtle animations

Avoid unnecessary visual effects.

---

## JavaScript Standards

* Keep JavaScript modular.
* Avoid inline scripts.
* Avoid global variables.
* Comment only when necessary.
* Prioritize readability.

---

## Website Structure

The website should include:

* Home
* About
* Services
* Portfolio
* Reviews
* Blog
* Contact
* Privacy Policy
* Terms
* 404 Page

---

## Design Goals

The design should communicate:

* Trust
* Professionalism
* Expertise
* Simplicity
* Credibility

Use a modern layout with generous spacing and clear visual hierarchy.

---

## Blog

The blog is a core feature of this website.

Blog posts will focus on:

* Meta updates
* Facebook
* Instagram
* Business Manager
* Social media security
* Recovery guides
* Digital marketing
* Google Knowledge Panel
* AI tools

Build the blog structure so it is easy to add new articles later.

---

## SEO

Optimize every page for search engines.

Include:

* Descriptive titles
* Meta descriptions
* Internal linking
* Image alt text
* Structured data placeholders
* Clean URLs

---

## Accessibility

Follow accessibility best practices.

Support:

* Keyboard navigation
* Visible focus states
* Color contrast
* Proper labels
* ARIA attributes where appropriate

---

## Performance

Prioritize:

* Fast loading
* Optimized images
* Minimal CSS
* Minimal JavaScript
* Clean markup

---

## Code Quality

Generate production-ready code.

Keep files organized.

Do not create unnecessary files.

Preserve existing architecture unless a better approach is explicitly requested.

Always maintain consistency across the project.
