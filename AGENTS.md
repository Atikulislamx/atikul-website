# AI Development Guide

This document defines how GitHub Copilot Agent should build and maintain this project.

The AI should always prioritize consistency, readability, maintainability, accessibility, SEO, and performance over unnecessary complexity.

---

# Project Overview

Project Name:

ATIKUL WEBSITE

Owner:

Atikul Islam Rabbi

Website Type:

Official Personal Website

Purpose:

A professional website representing Atikul Islam Rabbi, showcasing expertise in social media security, publishing educational blog content, presenting services and case studies, and providing a trustworthy online presence.

---

# Core Principles

The AI should always generate code that is:

* Clean
* Modular
* Well documented
* Responsive
* SEO friendly
* Accessible
* Fast
* Production ready

Avoid unnecessary complexity.

Do not generate experimental solutions when a stable approach exists.

---

# Technology

Always use:

* HTML5
* CSS3
* Vanilla JavaScript (ES6)

Do NOT use:

* React
* Vue
* Angular
* Next.js
* TypeScript
* jQuery
* Bootstrap
* Tailwind CSS
* Node.js
* npm packages unless explicitly requested

The website should work directly after opening the HTML files or when hosted on GitHub Pages.

# CSS Guidelines

Organize CSS logically.

Maintain readable formatting.

Prefer reusable utility classes.

Avoid duplicated CSS.

Responsive design is required.

Animations should remain subtle.

Avoid excessive visual effects.

# JavaScript Guidelines

JavaScript should be modular.

Separate logic whenever practical.

Do not use inline JavaScript.

Avoid global variables whenever possible.

Code should remain readable.

Comment complex logic.
